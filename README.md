# Face-Recognition

This is a website that uses Clarifai AI to performs face recognition on images.

## Tech Stack

**Client:** React 

**Server:** Node, Express

**DB:** Postgresql

**Authentication:** JWT

## Installation

Installation with npm

```
  git clone this repository
  npm install
  cd facerecognition
  npm start
```

  in a second terminal: 
``` 
  cd Backend
  npm start
```
